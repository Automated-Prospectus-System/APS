🛠️ TROUBLESHOOTING:

  "MySQL not running?"
  → sudo service mysql start (Linux)
  → brew services start mysql (macOS)
  
  "Can't connect to database?"
  → Check credentials in .env
  → Verify MySQL user created
  → Run: mysql -u aps_user -p aps_system
  
  "No programs in database?"
  → Place PDFs in data_extraction/prospectuses/
  → Run: python3 database/data_loader.py
  → Check: SELECT COUNT(*) FROM programs;
  
  For more help → See TROUBLESHOOTING.md
