# 📡 API EXAMPLES

## Get All Universities
```bash

```

## Search Universities
```bash
curl "http://localhost:5000/api/universities?search=National"
```

## Get All Programs
```bash
curl http://localhost:5000/api/programs
```

## Filter Programs
```bash
curl "http://localhost:5000/api/programs?field=Science%20%26%20Technology"
```

## Check Eligibility
```bash
curl -X POST http://localhost:5000/api/eligibility \
  -H "Content-Type: application/json" \
  -d '{
    "subjects": [
      {"subject": "Mathematics", "grade": "A"},
      {"subject": "English", "grade": "B"},
      {"subject": "Physical Science", "grade": "C"}
    ]
  }'
```

# 🐛 TROUBLESHOOTING

### Backend won't start
```bash
# Check if port is in use
lsof -i :5000  # macOS/Linux
netstat -ano | findstr :5000  # Windows

# Try a different port
python app.py  # Edit port in app.py if needed
```

### Frontend API calls failing
- Check backend is running: `http://localhost:5000/api/health`
- Check browser console (F12) for errors
- Verify CORS is enabled in backend
- Check frontend is using correct API URL

### Python modules not found
```bash
# Reinstall dependencies
pip uninstall -r requirements.txt
pip install -r requirements.txt
```
