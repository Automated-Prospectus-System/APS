Authentication Flow

User Visit Site
    ↓
Try to Access Protected Page
    ↓
protectPage() checks isAuthenticated()
    ↓ (Not authenticated)
Redirect to login.html
    ↓
User Enters Credentials
    ↓
AuthManager.login(username, password)
    ↓ (Valid)
Create Session in sessionStorage
    ↓
Redirect to Requested Page or home.html
    ↓
User Now Has Access to All Protected Pages
    ↓
User Clicks Logout
    ↓
Session Cleared
    ↓
Redirect to login.html
