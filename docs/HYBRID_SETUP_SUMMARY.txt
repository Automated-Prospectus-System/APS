# ⚡ Hybrid MySQL Setup - Quick Reference

## 📋 What Changed?

Your system now **requires MySQL** but **auto-initializes with 1 command**.

| Before | After |
|--------|-------|
| MySQL optional (fallback to JSON) | MySQL is REQUIRED |
| Setup: 10+ manual steps | Setup: 3 steps + 1 command |
| Silent failures with JSON data | Hard fails with clear guidance |

---

## 🚀 Setup (3 Steps Only)

### 1. Install MySQL
```bash
# Ubuntu/Debian
sudo apt-get install mysql-server mysql-client
sudo service mysql start

# macOS
brew install mysql
brew services start mysql
```

### 2. Configure .env
```bash
cp .env.example .env
# Edit .env - update DB_PASSWORD if needed
```

### 3. Initialize Database
```bash
python3 database/init_mysql.py
```

That's it! Database is now ready with:
- ✅ 6 universities pre-loaded
- ✅ 20 programs pre-loaded
- ✅ Proper schema and indexes

## 🔧 Troubleshooting

| Error | Solution |
|-------|----------|
| "MySQL connection failed" | Run `python3 database/init_mysql.py` |
| "Access denied" | Check .env credentials match MySQL root password |
| "Database already exists" | That's OK! Script will reuse it |
| "User already exists" | That's OK! Script will use existing user |
